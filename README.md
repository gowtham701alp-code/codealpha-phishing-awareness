# Phishing Awareness Training

## Overview

This project was completed as part of a Cybersecurity Internship. It focuses on educating users about phishing attacks, how cybercriminals use social engineering techniques, and the best practices to identify and prevent phishing attempts.

## Objective

The objective of this project is to create an engaging cybersecurity awareness presentation that helps users:

* Understand what phishing is.
* Recognize different types of phishing attacks.
* Identify phishing emails and fake websites.
* Learn common social engineering techniques.
* Follow cybersecurity best practices to stay protected.
* Improve awareness through a short interactive quiz.

## Project Contents

* **Phishing_Awareness_Training.pptx** – PowerPoint presentation.
* **Phishing_Awareness_Training.pdf** – PDF version of the presentation.
* **README.md** – Project documentation.

## Topics Covered

* Introduction to Phishing
* Types of Phishing Attacks
* Phishing Statistics
* Anatomy of a Phishing Email
* Fake Login Page Examples
* Social Engineering Techniques
* Prevention and Protection Methods
* Multi-Factor Authentication (MFA)
* Incident Response
* Interactive Quiz
* Key Takeaways

Through this project, I gained knowledge about:

* Modern phishing techniques

## Conclusion

Phishing remains one of the most common cybersecurity threats. User awareness is the first line of defense against these attacks. This project demonstrates the importance of recognizing phishing attempts and following safe online practices to protect personal and organizational data.
